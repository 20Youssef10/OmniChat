# OmniChat AI User Guide

Welcome to OmniChat AI! This guide will help you get started with the features of the application.

## Getting Started

1.  **Sign In**: Click the "Sign In" button in the sidebar. You can use Google, Email, or continue as a Guest (data is local only).
2.  **Select a Model**: Use the dropdown in the top header to choose an AI model (e.g., Gemini 2.5 Flash, GPT-4o).
3.  **Start Chatting**: Type your message in the input box at the bottom.

## Features

### Chat & Media
-   **Text**: Standard chat.
-   **Images**: Click the paperclip icon to upload images for analysis.
-   **Voice**: Click the microphone to speak. The AI can also read responses aloud.
-   **Image Generation**: Select "DALL-E 3" or "Gemini Pro Image" to generate visuals.
-   **Video Generation**: Select "Veo" models to generate short videos.

### Power Tools
-   **Slash Commands**: Type `/` in the input box to see quick actions like `/summarize` or `/analyze`.
-   **Comparison Mode**: Click the "Compare" toggle in the model dropdown to run two models side-by-side.
-   **Search**: Press `Cmd+K` (or `Ctrl+K`) to open the global search.

### Organization
-   **Projects**: Group chats into projects via the sidebar folder icon.
-   **Workspaces**: Create shared workspaces for team collaboration.

### Personalization
-   **Personas**: Switch the AI's personality (e.g., Coder, Creative Writer) via the User icon.
-   **Memories**: Add facts about yourself in the Brain icon menu so the AI remembers them.

## Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| `Cmd/Ctrl + K` | Open Search |
| `Enter` | Send Message |
| `Shift + Enter` | New Line |
