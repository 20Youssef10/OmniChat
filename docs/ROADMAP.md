# Product Roadmap

## Phase 1: Mobile & Access (Q2)
- [ ] **PWA Enhancement**: Improved offline caching strategies, Service Worker background sync, and "Add to Home Screen" prompts.
- [ ] **Mobile Native Apps**: Wrappers (Capacitor/Ionic) for iOS App Store and Google Play Store release.
- [ ] **Browser Extension**: Chrome/Edge extension for quick sidebar access to OmniChat context while browsing.

## Phase 2: Intelligence & Data (Q3)
- [ ] **Model Fine-Tuning**: Allow enterprise users to fine-tune Gemini/OpenAI models on their own datasets.
- [ ] **Advanced Analytics**: Machine Learning insights on user chat patterns (sentiment analysis trends).
- [ ] **Plugin Marketplace**: Community-driven extensions for specific workflows (e.g., Jira integration, Figma export).

## Phase 3: Enterprise & Scale (Q4)
- [ ] **Desktop Application**: Electron-based desktop app with system-wide shortcut keys and OS integration.
- [ ] **White-Labeling**: Theming engine allowing businesses to rebrand OmniChat with their own logo and colors.
- [ ] **Enterprise API**: Higher rate limits, dedicated support channels, and SLA guarantees.
- [ ] **SSO Integration**: SAML/OIDC support for corporate login (Okta, Azure AD).
